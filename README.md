Bu projede:                                                                                                  
-ORM aracı olarak Entity Framework kullanıldı.                                                                                
-DbFirst yaklaşımı ile projeye veri tabanı eklendi.
-Entity Framework methotları kullanılarak veri tabanına Rehberler ve Lokasyonlar eklendi.
